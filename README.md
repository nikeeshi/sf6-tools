# SF6-tools

スト6(ストリートファイター6)関連のツール集。各ツールはサブディレクトリで管理する。

## プロジェクト一覧

- [frame-meter](frame-meter/README.md) — フレームメーター風の図を作る解説用UIツール
- [combo-tree](combo-tree/README.md) — コンボルートを木構造で管理する自分用ツール
