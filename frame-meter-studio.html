Frame Meter Studio

格ゲー風フレームメーターUI

スト6の対戦画面にあるフレームメーター(発生・持続・硬直をバーで示す表示)の見た目だけを再現するツールです。 上の数値はすべて説明用のサンプルで、特定キャラクターの実数値ではありません。

Cursor
F--

ルーラー付近をクリック／ドラッグするとカーソル(白線)を好きなフレームに固定できます。バーにマウスを乗せると内訳がツールチップで出ます。 自分でチャートを作る場合はこのファイル内 DATASETS を編集してください。1レーン = { id, label, sub, startFrame, segments, connect }、 分岐は connect: { from: '親レーンid', atFrame: 分岐フレーム } を指定すると、その位置から下に配線されます。
