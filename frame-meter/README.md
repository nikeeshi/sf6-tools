# frame-meter-studio

発生・持続・硬直などをフレーム単位の色付きバーで表示する、フレームメーター風の図を作るためのプロトタイプ。

- 単一のHTMLファイル(SVG + Vanilla JS、外部ライブラリなし)。ブラウザで開くだけで動く
- 1フレーム=2:3比率のセルとして横に並べて表示
- 択・派生(投げ/打撃/遅らせ投げ など)を、途中のフレームから下に分岐させて表示できる
- チャート上をクリック/ドラッグすると、フレーム位置を示すカーソルを固定できる
- ファイル内の `DATASETS` を編集することで、自分でチャートを追加・編集できる

数値はすべて説明用のサンプルで、特定キャラクターの実数値ではない。
